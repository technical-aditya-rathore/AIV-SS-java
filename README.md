Array Input Validation & Sorting System (Java)
--------------------------------------------------------------------------------------------------------------------------------

A robust Java console application that:

Takes validated numeric input

Handles integers and decimals

Sorts the array using Bubble Sort

Prints results cleanly based on data type

This project focuses heavily on input validation and edge cases.
--------------------------------------------------------------------------------------------------------------------------

 Features

Strict validation for:

Integers

Decimal numbers

Negative values

Prevents invalid numeric formats

Custom sorting logic

Clean formatted output

Unit testing included

----------------------------------------------------------------------------------------------------------------------------
Core Components

checkString() → integer validation

checkStringWithDouble() → decimal validation

InitArray() → safe array initialization

Sort() → bubble sort implementation

PrintArray() → formatted output
